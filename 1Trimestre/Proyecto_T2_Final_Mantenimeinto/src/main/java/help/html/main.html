Ayuda de la App Incidencias

Esta Aplicacion esta creada para el IES Segundo de Chomon

Organiza todas las incidencias posibles en un instituto

Esta app ha sido creada para organizar un poco mejor el instituo en cuanto problemas que puedan surgir durante el horario lectivo, y llegara automaticamente a los tecnicos para poder resolver los dichos problemas

Para abrir la ayuda pulde F1 en la pantalla de incidencias y profesores
